This plugin provides customizable insults from F*ck Off As A Service via shortcodes.

== Description ==
Do you ever wish you had more profanity in your posts? Install this plugin, and you can tell off all sorts of people in all sorts of ways! WP-FOAAS gives you the API-powered power to tell people to f*ck off, dynamically!

This plugin connects you to the API provided by http://www.foaas.com/, but don\'t blame them, they actually had nothing to do with it.

== Installation ==
1. Upload WP-FOAAS to the /wp-content/plugins/ directory.
2. Activate the plugin through the \'Plugins\' menu in WordPress.
3. Navigate Settings ->WP_FOAAS Settings for hints on how to construct shortcodes.
4. Enjoy!

== Frequently Asked Questions ==
1. **How do I do multi-word names?**
All you need to do is put multi-word attributes in quotes, e.g. [foaas command=cool name=\'your mom\']
